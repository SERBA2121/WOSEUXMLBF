# WOSEUXMLBF
LBF - Lags But Flagship,The firmware is designed with an emphasis on flagship features and visuals, but you will still be able to use 120 Hz relatively comfortably, even with advanced textures.

### Flashing guide
- Unpack archive on C: drive,
- Start flash_all.bat,
- Format data via recovery,
- Reboot,


### Rom features
- Global 15T pro base WOSEUXM3.0.0.4- 
- Added advanced textures
- Disabled Control center opening\closing animation
- Added depth feature
- Depth feature without restrictions
- Disabled blur in opened folder
- Disabled blur when opening\closing app
- Disabled blur in recents
- Debloat
- Devicelevellist 111
- Kigurumi wallpapers
